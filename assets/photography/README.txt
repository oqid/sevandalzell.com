Drop your Fujifilm shots here, named 01.jpg, 02.jpg, 03.jpg etc.

Then in index.html, find the photo-grid section and replace each
placeholder <figure> with:

  <figure class="photo-item">
    <img src="assets/photography/01.jpg" alt="Describe the shot">
  </figure>

Keep images reasonably sized (under ~500kb each, longest edge ~1600px)
so the page loads fast — Squoosh.app is a good free tool for this.
